# Support

## Report a defect

Open a [bug report](https://github.com/ejupi-djenis30/IntegraDraw/issues/new?template=bug_report.yml)
with the implementation you used, the exact expression and interval, the expected result and the
smallest steps that reproduce the problem. Include your Java or browser version when it matters.

## Propose an improvement

Use the [feature request](https://github.com/ejupi-djenis30/IntegraDraw/issues/new?template=feature_request.yml)
and describe the mathematical or usability problem first. Please keep separate ideas in separate
issues.

## Report a security problem

Do not open a public issue. Follow the [security policy](SECURITY.md) and use
[private vulnerability reporting](https://github.com/ejupi-djenis30/IntegraDraw/security/advisories/new).

IntegraDraw is an open-source project rather than a managed service. Response times are not
guaranteed, but complete, reproducible reports are prioritized.
